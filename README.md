# Cybersecurity Framework for a Small Business

A complete cybersecurity framework designed for small businesses, covering governance, policies, incident response, disaster recovery, business continuity, and data protection.

## Features

- Acceptable Use Policy
- Password Policy
- Access Control
- Data Protection
- Risk Assessment
- Incident Response Plan
- Disaster Recovery Plan
- Business Continuity Plan
- Vendor Security Policy
- Logging and Monitoring
- Compliance Guidelines
- Security Awareness Training

## Framework

This project follows industry best practices including:

- NIST Cybersecurity Framework
- ISO 27001
- CIS Critical Security Controls
